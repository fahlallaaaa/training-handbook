# Choosing Tech Stack

Date: 2024-03-30

## Status

Accepted

## Context

ทุก ๆ ทีมถนัดพวก Framework ที่เป็น JavaScript และถนัดในการแยก Frontend กับ Backend

## Decision

- Frontend เลือกใช้ Next.js ส่วนตัว CSS ใช้ Tailwind
- Backend ใช้ Node.js กับ Express
- Database ใช้ MongoDB
- Infra ใช้ของ AWS
